"Gitting" the Project

To work on the Recursive Thinking website project, youâ€™ll need to transfer the project files from the Recursive Thinking GitHub repository to your computer.

This will be accomplished by running the command git clone in your terminal/bash/windows CLI.

There are two different repositiories which you'll need to clone from the Recursive Thinking GitHub Page.

    Front End:
  • recursive-thinking-website

How to Clone a repository from GitHub

  1. First, from your terminal, run git --version. This command should return the version number of git that you downloaded and installed. If it doesn't, you need to repeat the download and installation of Git. (See Get PreReqs)

    If this command returns a version of Git that your machine is running, you are all set to continue!

  2. Second, make sure you are in the directory that you would like the repository to reside. For Example, if you have a web projects directory - just make sure to navigate inside that folder before you start running the git commands. This is important because when the git commands are executed, the repository will be located at the location of the current terminal.

  3. Third - go to the the Recursive Thinking Website Github Page.

    On the website's main page, under the status/statistics bar - and to the far right of the screen, you should see a green "Clone or Download" Button.

    When this button is clicked, you'll see a modal pop up with an address, and most likely titled "Clone with Https". This address will, depending on your configuration, be either in https:// or ssh. Since https:// is the default protocol (and using ssh requires special setup), we'll cover how to use only https:// in this section.

    The https:// address for the Recursive Thinking Website is: https://github.com/RecursiveThinking/recursive_thinking_website.git

    Once you have this address, go back to your terminal, and command: git clone <https://address>

    for the Recursive Thinking Website, this would look like:

    git clone https://github.com/RecursiveThinking/recursive_thinking_website.git

    After executing this command, you should see some additional status reports in your terminal, indicating the current progress, as well as the success/failure of this command. Of course, we are looking for an indcator that this command was successful. Additionally, if the command was successful, you should see a folder named "recursive-thinking-website" in the directory where you executed the git clone command.

    If you don't get a successful clone, and you don't see the folder in your directory, please re-run these steps until you do.

  4. Last, once the repository has been cloned locally, you can take the steps necessary to setup and run the website.

    The Recursive Thinking website uses Webpack to compile all of the separate HTML, CSS and JavaScript files into a single page application.

    Webpack is a part of Node Package Manager (NPM) which allows users from all over to share "building blocks" of JavaScript code.

    In the Recursive Thinking Website folder, you'll see a file called package.json. package.json contains all the npm modules needed to run the website. To install these modules, make sure the terminal is in the recursive_thinking_website folder and run the Node command: npm install.

    If all the required node packages installed successfully, you should now see a "node_modules" folder in the recursive_thinking_website directory. If you see this folder, the modules are successfully installed, if not please repeat the last step.

    Once you have the node_modules folder, you just need to run one more command in your recursive_thinking_website directory - npm run start

    When this command is executed, there will be alot of output in the terminal, and after a few seconds, the site will build in Webpack, then it will open a tab in your default browser, at localhost://8000, and will load the website.
