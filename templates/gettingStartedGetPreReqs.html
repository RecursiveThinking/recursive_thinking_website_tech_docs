Get Prerequisites

Getting closer! To be able to make view and make changes to this project, you will need to install some commonplace software. These will be used for this and most all of your other future development and design projects.

But what's it for?

Your IDE is where you write and edit code. There are a few different options. If you are just getting started, it's recommended to pick one from below and learn on it. You can always switch to a difference IDE later when you have more experience. VSCode is a great one to start with.

Node.js will be needed to install helper modules (libraries) for JavaScript.

Git is a version control system to help track changes to the code for our project. With many people working on it, often at the same time, we need a way to save versions of the code so that we don't accidentally overwrite something critical.

Node.js

(Make sure you download the correct version for your OS)

  • Download Node.js

  • You can check if Node and NPM (bundled with Node) are installed by opening your terminal/command line and typing node -v or npm -v and hitting 'enter'. It will return a version number below if correctly installed.

Git

(Make sure you download the correct version for your OS)

  • Download Git

  • **Windows Users: Note that Mac (OSX) has a built in terminal that supports bash. Windows, by default does not. When you install Git on Windows, it should by default, install Git Bash, which is the terminal most often used to run git commands. Windows users can check for a Git Bash installation simply by searching for it in your start bar or windows explorer.
